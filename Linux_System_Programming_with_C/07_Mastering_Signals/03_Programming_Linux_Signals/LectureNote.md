# Lecture Note



## 試す

simple-handler を実行して、別の Terminal から signal を送って受け取らせる。  



## 使うコマンド

### pgrep ExecuteName

ExecuteName で ProcessID を取得する

### kill -INT ProcessID

ProcessID に SIGINT を送る

### kill -TERM ProcessID

ProcessID に SIGTERM を送る

### kill -KILL ProcessID

ProcessID に SIGKILL を送る
